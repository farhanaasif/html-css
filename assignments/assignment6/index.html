DataTables Plug-in

DataTables is a free open source, highly flexible, and light weight plug-in for the jQuery Javascript library. It's zero-configuration implementation makes it distinct from other plug-ins. Works effectively for static and dynamic dataset.

It's main features include but not limited to:

Features highlights:

  • Pagination
  • Multi-column sorting
  • Sorting filtering
  • Instant search
  • Easy to theme with jquery UI, Bootstrap
  • Professional looking default theme
  • Ajax auto loading of data
  • Highlights Sorting column(s)
  • Scrolling options for table viewport
  • Smart handling of column widths
  • Provides a list of beautiful and professional styles
  • Has a style creator for customization
